# Terminal kanban
a terminal based `KanBan board using textual`.
helps you manage your `tasks from your terminal` with a clean UI & a keyboard-driven workflow.

# Features
- kanban style task management (boards & cards)
- fully keyboard driven
- runs entirely using the terminal
- lightweight and fast!
